Zong! WebApp

Notice: In order to run this web application in Eclipse, check the following points:

  • use the webapp.launch for starting, or provide the VM parameter -javaagent:"/absolute/path/to/your/eclipse/lombok.jar"=ECJ"
  • update the two absolute paths in webapp.launch. One points to the war directory of the project, the other one to the lombok.jar in your eclipse installation
  • delete the cache (i.e. the "gwt-unitCache" folder), if it already exists
